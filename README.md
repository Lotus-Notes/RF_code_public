# RF_code_public

📡 **RF & Microwave Engineering Utilities (MATLAB + Python)**

This repository contains a curated collection of **public, non-proprietary RF and microwave engineering scripts** developed for analysis, simulation, and design exploration.  

---

## 📌 Repository Scope

The scripts in this repository are intended to support:

- RF / microwave circuit analysis
- Network parameter manipulation (S, Z, Y, ABCD)
- Matching and impedance transformations
- Frequency-sweep studies
- Design-space exploration and visualization
- Quick sanity-checks during RF design work

---

## 🛠️ Languages & Tools

- **Python**
  - `numpy`
  - `scipy`
  - `matplotlib`
  - `scikit-rf`
  - `pandas` (where applicable)

- **MATLAB**
  - RF Toolbox–compatible workflows
  - Vectorized numerical analysis
  - Publication-quality plotting

---

## 📂 Example Topics Covered

> (Exact contents evolve over time as scripts are added/refined.)

- Impedance and admittance conversions
- Stability factor calculations (K, μ)
- Matching network exploration
- Frequency-dependent parameter sweeps
- Load-pull style numerical experiments (non-measured)
- Visualization utilities for RF metrics

---
